- weui 结合小程序开发
  weui 是微信like 的界面解决方案，适用于公众号，小程序快速界面开发
  1. 引入weui.wxss 文件
  2. 来到WEUI 的组件官网(https://weui.io/) 要用那个用那个覆盖了70% 的常用组件

  input 视图
