- 用闭包来构建私有属性和方法
    函数内嵌套函数，就会形成作用域的嵌套， 形成闭包
    因为闭包内部的函数， 会保存对外部函数内的变量的持久引用，所有外部函数的变量生命周期就变长了，
    闭包： 函数在运行时， 可以引用到定义时的上下文环境的变量

- 快速构建闭包
    使用立即执行函数 + return 一个函数  （快捷构造出闭包）
    构建类，
    静态属性和方法只在类上调用
